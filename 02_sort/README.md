## 内容 

Gtk.Entryに入力したテキストをGtk.StringListに項目として追加する。Gtk.DropDownの選択肢として表示する時に、Gtk.StringListをソートして表示する。


![pic](../data/pyt_gtk4_dropdown02.webp)

※ 画像中で項目は、ccc1 - bbb2 - aaa3 - ddd4の順に追加したが、Gtk.DropDownの選択肢はアルファベット順に並んでいる。
